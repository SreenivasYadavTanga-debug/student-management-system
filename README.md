# Student Management System

A Student Management System built using HTML, CSS and JavaScript.

## Features

- Add Student
- Delete Student
- Search Student
- Upload Student Photo
- Export CSV
- Local Storage Support
- Student Counter

## Technologies Used

- HTML5
- CSS3
- JavaScript
